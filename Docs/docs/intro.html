class Vector.Vector(x, y)¶
static dist_between_vectors(v1, v2)¶

Returns the distance between two passed in Vectors. Requires slow sqrt operation.

dot(vector)¶

Returns a new Vector equal to the product of self and passed in Vector - does not modify self.

length¶

Returns the length of the Vector. Requires slow sqrt operation.

normalize()¶

Normalises self x and y values between 0 and 1 - modifies self. Will not operate if Vector length is 0.

static normalize_vector(vector)¶

Returns a new Vector equal to normalisd self x and y values - does not modify self. Will not operate if Vector length is 0.

perp¶

Returns the Vector perpendicular to self - does not modify self.

sign(vector)¶

Specifies if self is clockwise or anti-clockwise from the passed in Vector - does not modify self. Returns -1 if anticlockwise, 1 if clockwise.

static sq_dist_between_vectors(v1, v2)¶

Returns the square distance between two passed in Vectors. Preferred by comparison of distance between multiple Vectors, as slow sqrt operation is not required.

sq_length¶

Returns the square length of the Vector. Preferred for comparison between Vectors as slow sqrt operation is not needed.

truncate(max_speed)¶

Clamps the Vector x / y values to the passd in maximum value - modifies self.

zero()¶

Resets self x and y values to 0 - modifies self.
