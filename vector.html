Vector¶

class Vector.Vector(x=0, y=0)¶

2D Vector used throughout the program. Defines operator overloads and various helper functions. This class also doubles as a Point2D. Most methods are invoked on and will modify a Vector; however, many have static equivalents which will not modify the passed in Vector.

Initializes x and y components of the Vector.

length¶

Returns the length of the Vector. Uses a sqrt operation so should not be used for comparison between Vectors.

normalize()¶

Normalises the x and y values of the current Vector to between 0 and 1. Will not operate if the Vector length is 0.

perp¶

Returns a new Vector facing perpendicular to the current Vector.

sq_length¶

Returns the square length of the Vector. No sqrt operation used so this is preferred for comparison between Vectors.

truncate(max)¶
Clamps the current Vector to the passed in maximum value.
If the current Vector exceeds the max length, it is normalised and multiplied by the max length.
Parameters:max (float) – The value to clamp the current Vector to.
zero()¶

Resets the x and y values of the current Vector to 0.
